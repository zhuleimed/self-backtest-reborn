# Report Builder Skill

将回测结果输出为可视化的报告文件。

## 输出文件

| 文件 | 格式 | 内容 |
|------|------|------|
| `*_trade_records.csv` | CSV | 每笔交易的完整明细 |
| `*_account_curve.csv` | CSV | 账户每日资金曲线（含基准对比）|
| `*_metrics.csv` | CSV | 绩效指标汇总 |
| `*_returns.png` | PNG | 策略 vs 基准累积收益率对比图 |

## 绩效指标

核心指标（打印到控制台并保存为 CSV）：

| 分类 | 指标 | 说明 |
|------|------|------|
| 收益 | 总收益率 | 期末总资产 / 初始资金 - 1 |
| 收益 | 年化收益率 | 252 个交易日年化 |
| 收益 | 超额收益 | 策略收益率 - 基准收益率 |
| 风险 | 最大回撤 | 累积收益率曲线峰谷最大跌幅 |
| 风险调整 | 夏普比率 | (日收益率 - 无风险利率) / 日收益率标准差 * sqrt(252) |
| 风险调整 | Sortino比率 | 同上，分母改为下行波动率 |
| 风险调整 | Calmar比率 | 年化收益率 / |最大回撤| |
| 统计 | 胜率(日) | 正收益日数 / 总交易天数 |
| 统计 | 止损/止盈次数 | 各风控触发次数 |
| 统计 | 盈亏比 | 总盈利 / 总亏损 |

## 输出目录
所有文件保存到 `output/` 目录。
